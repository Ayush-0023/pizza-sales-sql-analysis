# 🍕 Pizza Sales SQL Analysis Project

This project focuses on analyzing **Pizza Hut sales data using SQL (MySQL)** to extract meaningful business insights from transactional datasets.

The analysis covers **sales performance, revenue trends, peak order timings, category contribution, and product-level rankings** using advanced SQL concepts.

---

## 📌 Project Objective
The objective of this project is to use SQL to solve real-world business questions related to pizza sales and customer ordering behavior.

This project helps answer:
- How much total revenue was generated?
- Which pizzas generate the highest revenue?
- What are the busiest ordering hours?
- Which pizza categories contribute the most?
- How does cumulative revenue grow over time?
- What are the top pizzas within each category?

---

## 📂 Dataset Tables
The project uses imported transactional data across the following tables:

- **orders**
- **order_details**
- **pizzas**
- **pizza_types**

---

## 📊 Business Questions Solved
### Basic
1. Retrieve total number of orders
2. Calculate total revenue
3. Identify highest-priced pizza
4. Find most common pizza size
5. Top 5 most ordered pizzas

### Intermediate
6. Category-wise quantity sold
7. Hourly order distribution
8. Category distribution
9. Average pizzas sold per day
10. Top 3 pizzas by revenue

### Advanced
11. Revenue contribution percentage
12. Cumulative revenue over time
13. Top 3 pizzas by revenue per category

---

## 💡 Key Insights
- Identified top-performing pizzas driving maximum revenue
- Found peak customer order hours for operational optimization
- Measured category contribution to total sales
- Analyzed cumulative revenue growth trends
- Ranked best-selling pizzas within each category
